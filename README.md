# Controle Fiscal TechStore - Sistema de NF-e e Livro Caixa

Este projeto é a materialização do Documento de Visão de um sistema desktop para controle fiscal, desenvolvido para auxiliar pequenos comércios na gestão de suas compras, vendas e despesas.

## 🛠️ Tecnologias Utilizadas
* **Python 3**
* **CustomTkinter** (Interface Gráfica)
* **SQLite3** (Banco de Dados Local)
* **Pytest** (Testes Automatizados - TDD)

## 🚀 Como executar o projeto
1. Instale as dependências: `pip install customtkinter pytest`
2. Inicialize o banco de dados: `python database.py`
3. Execute o sistema: `python app.py` (Credenciais: admin / admin)
4. Para rodar a suíte de testes: `pytest test_sistema.py -v`
